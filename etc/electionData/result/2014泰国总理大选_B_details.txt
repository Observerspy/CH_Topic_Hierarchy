﻿==简介
当地时间2月2日，泰国大选投票如期举行，但一些投票站因反政府示威民众阻挠而未能顺利展开投票。图为泰国总理英拉在曼谷投票。[1]

1	背景	2	0	[]
自从2006年2月反他信力量因西纳瓦集团股份出售案走上街头以来，两大阵营的对峙似乎已经陷入了一个议会斗争-街头政治-权威干预的怪圈：如果进行选举，必然是占选民多数的支持他信阵营获胜，反他信阵营就会进行抵制；不进行选举甚至恢复军政统治，支持他信阵营也会进行抵制。无论哪一个阵营的代表者组织政府，另一方都会采取街头政治的方式表示反对，靠国王这一最高权威出面获得暂时的平静，不久后又会掀起新一轮动荡。[2]
2月2日，泰国议会下议院选举如期举行。在历经3个多月的动荡后，这次民主程序的启动，能否遏制住日趋激烈的政治对抗态势，不仅决定着英拉政府的执政寿命，也决定着泰国能否重归相对平静的政治和社会运转周期。[3]

2	大选前期	2	0	[]
自2013年10月31日开始，泰反政府群体的抗议示威活动已超过3个月。分析人士指，由特赦条例激起不少民众不满发端，最终导致了英拉解散国会下议院，重新举行大选这一结果。否则，其做满剩余不到两年任期当无大碍。[4]
实际上，2013年11月11日国会上议院否决特赦条例后，不少民众本已平息了对当局不满之情绪。但反政府群体抓住了民众情绪不利于政府之良机，顺势将反对特赦条例的初衷，一步一步升级为迫使英拉解散国会，再发展到迫使英拉辞职。反政府群体步步紧逼之下，英拉在貌似退让中一直坚持着自己的防线至今。[4]
2013年12月9日，迫于反对派示威等政治压力，泰国总理英拉发表了电视讲话，宣布解散国会下议院。英拉在讲话中表示，解散国会是为了在危机时刻将权利还给人民。英拉希望所有党派都能参加即将到来的大选，政府将作为看守政府继续执政至大选之日。其后，看守政府决定将在2月2日举行大选。[5]
英拉在宣布解散国会下议院后，反政府群体举行了两次号称有数百万人参加的大规模抗议示威活动，要求英拉及其看守政府辞职，设立“人民议会”，由国王陛下指定临时总理，由“人民议会”推举贤能之士组成临时政府，待一切改革规划完成后，再举行国会下议院选举。[4]
英拉则始终坚持反政府群体的诉求于法无据，坚不辞职，并提出“先选举、后改革”、或选举与改革同步进行的主张；但反政府群体概而拒之，最后使出“封锁曼谷”之举，但仍未能动摇英拉坚不辞职之防线。[4]
1月中旬，泰选委会建议看守政府推迟选举日程，看守政府一直未有回应，选委会遂请宪庭释宪；泰宪法法庭1月24日以8：0的表决结果裁定，在明知局势可能给国家造成重大损害的情况下，可以重新确定大选日期；该庭还以7：1的表决结果，要求英拉与选委会共同协商出新的选举日期。[4]
截止目前，全国375个选区中已经有47个选区受到影响，并被迫关闭部分投票站。1日傍晚针对选举而发生的流血冲突事件让泰国政府排除万难极力推动的这次选举的结果越来越不被外界看好，而选举日之后的政局将走向也让人疑虑重重。[6]

3	封锁曼谷	2	0	[]
另一方面，在“封锁曼谷”街头示威行动继续的同时，泰国反对派表示将抵制大选。反对派称，为保证选举的民主性，并防止下任政府滥用权力，在大选前必须进行改革。[5]
1月26日，泰国大选提前投票正式开始，然而由于反政府示威者的阻挠，首都曼谷和南部多地的投票被迫取消。当天一早，示威者用链条封锁了通往多个投票站的道路或者直接将投票站团团围住。导致曼谷全市50个选区中，39个选区的提前投票工作被迫取消。据泰国选举委员会统计，在26日的提前投票中，有近44万人被反对派示威人群阻止投票。[5]
2月1日，大选前夕，泰民主党党魁、前总理阿披实称，他不会参加2月2日国会下议院选举的投票。他说，因为这次选举是违宪的，同时也不会实现民主制度下的选举目标。[5]

4	阿披实态度	2	0	[]
阿披实此前受访时表示，若大选延后，该党可能会参选，并敦促当局制定可行的改革路线图。他还呼吁各方通过谈判制定出一个路线图，设定一个合理的时间表，既可以启动国家的改革，又可以达致让各方都可以接受的大选时间。阿披实并不认同由非民选的“人民议会”管理这个国家的建议，称这不是民主党的要求；他同时也认为英拉已不具公信力，无法推进该国的改革进程。[5]
英拉当日要求阿披实就“本次选举违宪”作出解释。她说，看守政府执行的是2007年版宪法，特别是在选举规则的部分。她说，民主党其时也是根据前述法律规则而取得政权的。英拉表示，如果我们不尊重相关法律所设置的规则行事，我们将如何向国际社会解释，我们又如何来治理这个国家。[5]

5	大选花费	2	0	[]
本次大选预算在38亿泰铢左右，由于泰最大的反对党——民主党抵制，选举结果可能达不到国会最少需95%的议席才能召开新一届国会会议之最低要求。[5]

6	大选预测	2	0	[]
媒体也对大选来临时泰国局势走向做出各种预测，分析称，司法“政变”和街头冲突是最有可能发生的。以下是泰国局势走向五大预测：[5]
一、 “贫乏”的大选：不少选举候选人被示威所阻，未能完成登记，即使大选如期举行，也可能无法选出足够议员。英拉在本次选举中能否保住总理宝座，还是未知数。但2013年底她通过不信任案考验，显示其仍有一定支持率。[5]
二、 司法“政变”：过去，泰国法庭曾介入政治，如解散政党或禁止政客参政，英拉政府有近220人在接受反贪腐委员调查，专家认为这是最有可能的结果。泰国家肃贪委员会1月28日举行会议，讨论了英拉在国家稻米典押政策实施过程中涉嫌失职，未能阻止给国家带来的巨额经济损失。[5]
该委员会决定将调查结果整理成文正式指控并启动弹劾英拉的程序。肃贪委将在下星期正式通知英拉接受调查，她可以在肃贪委启动进一步法律程序前为自己辩护。[5]
三、 街头冲突：由于反对派“封锁”曼谷，并强硬声言抵制大选，大选期间泰国街头难言平静。一方面，红衫军计划举行示威支持政府，一方面，看守政府已发出警告，要求反对派在期限内撤离，否则可能以动武形式组织军警夺回政府机关控制权。两派势力或引发大规模流血冲突。[5]
四、 武力镇压：2010年，泰国民主党政府军事镇压红衫军示威，造成90人死亡，1900人受伤，目前，对于在野民主党的反政府活动，英拉政府似乎在极力克制，避免重蹈覆辙。但如果一旦情势危急，严重影响大选正常进行，英拉政府可能会有所动作，进行强行“清场”。[5]
五、 军事政变：泰国陆军总司令巴育多次表明不愿发动“政变”，但也并未完全排除这种可能。不少观察员认为，一旦军方强势介入，将改变整体局势走向。军方的“政变”甚至会令亲他信的红衫军发动新骚动。[5]

7	英拉	2	0	[]
英拉政府软硬兼施，予以坚决回击：软的是撤销特赦法案，消除反对派口实；硬的是拒绝接受宪法法院违宪判决，依靠下院多数打掉反对派的不信任案，并坚决否定所谓“人民议会”的提议，称其有违宪政。为化解危机，英拉于2013年12月9日宣布解散国会，并将尽快举行选举。[2]
英拉坚持先选举后改革，而反对党坚持先改革后选举。最大反对党民主党甚至表示将不参加大选。英拉对自己领导的为泰党有信心，认为该党有望在选举中再次获得胜利。在2011年的大选中，为泰党赢得国会下议院500个议席中的265个。而民主党抵制大选，是预计如果参加大选，所获席位可能不到100个。截至目前，双方都坚持自己的立场，泰国政局也因此陷入僵局。[2]
受到此前政局压力的影响以及对选举日局势的担忧，英拉在当天的投票过程中表情相对严肃，她在离开时对记者表示，根据选举法的规定，自己不能回答任何有关政局的问题，但是希望能够再一次呼吁全国民众尊重民主制度，积极前往行使自己的投票权利。[6]
英拉表示：“今天对泰国人民来说是一个重要的行使权利的日子，我想邀请全国人民共同来行使权利，参加选举不仅是为了维护民主，同时也是行使公民权利的表现。”[6]
据泰国媒体报道，泰国国会下议院选举当地时间2日举行，泰国看守总理英拉当天为大选助威，结果却闹出乌龙：在工作人员引导下，把选区票和政党票投进相反的票箱。英拉稍后认错，泰国选委会称英拉的票仍旧有效，计票时会进行纠正。[7]
英拉于当地时间2日早8点10分左右，前往Klong Lamchiak 学校投票站投下选票，并呼吁全体选民行使自己的投票权。[7]
但媒体拍摄到的英拉进行投票的照片和视频显示，英拉将橘黄色的政党票投进了选区票箱，而将紫色的选区票投进了政党票箱。[7]
报道称，英拉在投票后将这一错误告知了选举委员会。而泰国看守政府副总理蓬贴·帖甘扎纳(Phongthep Thepkanjana)也表示，他进行投票的恰图恰(Chatuchak)投票点也出现类似的错误。[7]

8	颂猜态度	2	0	[]
颂猜当天早些时候说，原拟2月2日举行的大选应推迟3至4个月，以便让各政党之间有时间来解决其业已存在的分歧。他说，各政党在此期间可以举行会谈，以寻求各方都能接受的解决目前政治冲突之途径。[5]
颂猜说，社会目前普遍担心即使大选日程被推迟，反政府示威者也不会结束他们的反政府抗议集会。他说，如果各政党与看守政府在选举和改革的问题上设定一个条件，选委会愿充当调停人与反政府抗议领导人举行会谈，其抗议示威活动应可结束。[5]
颂猜还表示，如果看守政府坚持其大选必须在2月2日如期举行之立场，选委会也别无选择，即使在2月2日选举时可能发生暴力性质的流血冲突，也将谨遵相关选举法的规定如期举行大选。他说，选委会已作好寻求军方和警方帮助的准备，以确保大选期间的和平和秩序。[5]

9	投票过程	2	0	[]
泰国国会下议院选举当地时间2日早8时至下午3时举行，整个投票过程比较顺利。曼谷和泰国南部部分选区一些投票站由于示威者干扰而无法进行投票，但未发生严重冲突事件。泰国选举委员会在投票结束后发表声明说，因目前收集到的选票不完全，所以暂时无法公布选举结果。[7]

10	选民观点	2	0	[]
前来投票的本亚提（女）表示，虽然反政府示威者从上周日提前投票日起就采取了阻碍投票的行为，同时还呼吁民众不要投票，但是自己不会在意他们，因为自己的权利不会被他人的想法所左右。[6]
“我们来行使公民权利，不在乎（示威者），大家都是爱国的，但这是我们的权利，我们想来的话，没有人可以阻止我们。他们如果要来，我们会欢迎。”[6]
家住丁当区的选民纳荣表示，自己在该区出生和长大，从来没有遭遇过类似的事情，如果警察不愿意帮助选民驱散示威者的话，选民们就将自发组织起来抗议示威者，并夺回属于自己的投票权。[6]
纳荣说：“住在这里的民众要集合起来，我们要反抗，如果警察不愿意帮助主持公正，这个国家如果要这样分裂下去，我们就要求改变。我们从小住在这里，而他们正在侵犯我们的权利。”[6]

11	部署警察	2	0	[]
泰警方29日称，将部署20万警察在全泰77个府375个选区9.3万多个投票站，以维持2月2日大选投票期间的法律和秩序。[8]
泰国家警察局助理总监阿纳称，警方将避免使用武力而采用和平谈判的方式解决可能出现的反政府抗议活动。他说，从目前评估情况看，相信大选投票当天不会发生暴力冲突事件；但如果发生冲突事件，相信也只是在少数地区，警方已部署快速反应部队作好应对准备。[8]
泰看守政府劳工部长察霖29日称，应曼谷市政府要求，2月2日大选投票时警方将在曼谷各投票点部署1万警察，为前往投票的选民提供安全保障并防止有示威者阻扰投票。[8]
察霖说，为确保选举顺利进行，警方首先要保证票箱能够顺利送达各投票站，他还呼吁曼谷市民2月2日行使其投票权，称警方将确保投票选民的安全。[8]

12	示威者与反政府者	2	0	[]
愤怒的民众当天中午还一度突破警察防线，与反政府示威者几乎发生面对面冲突，来自南部洛坤府的示威者保安沙南表示，自己和示威者们一大早就从胜利纪念碑的集会场来到这里。虽然示威者们已经将投票站强行关闭，但是沙南认为，自己的做法并不是在阻碍选举，而是在用不同的方式催生改革之后更加完善的选举。[6]
“我们也是人民，我们从胜利纪念碑来这里声援示威者，我们手里也没有武器，可是他们冲过来侵犯我们，我们就要保护自己。我们没有阻碍选举，但我们要求先改革，等改革完了再进行选举。”[6]
中午11点半左右，在警察进行多次劝说之后，泰日体育馆的示威者们宣布撤离该投票站，返回集会场。而已经被关闭的投票站内的工作人员已经全部离开，投票工作无法再重新进行。[6]
反政府群体“君主立宪制下完整民主制度人民委员会”称，该群体不会阻挠2月2日的大选投票，但会在当天“完全”封锁曼谷，使整个曼谷变成步行街。该群体在泰南地区的支持者已围困当地一些邮局，阻止大选选票被送达该地区投票站。[5]
反政府群体领导人素贴等1日率领其支持者由曼谷隆披尼公园一路游行至曼谷唐人街等处，试图鼓动更多的民众反对看守政府“先选举、后改革”的抗议活动。[5]
当地时间2014年2月2日，泰国国会下议院选举投票开始举行。而就在此前一天，泰国反政府示威者与支持政府者在首都曼谷街头展开了枪战。共传出数十声枪击声和至少两起爆炸声。[9]
反他信阵营是以军队、官僚和部分城市中产阶级为主的精英阶层，挺他信阵营则是以东北部农民为主的草根阶层。前者是泰国传统精英政治体制的受益者，而后者则把他信执政期间所推行的政策视为其利益的代表。他信依靠占泰国人口约70%的农民的支持，在2001年和2005年两次赢得大选，其领导的泰爱泰党成为泰国历史上第一个“一党独大”的政党(国会议席比例高达75.4%)，打破了多年来泰国多党制精英政治的平衡。[2]
