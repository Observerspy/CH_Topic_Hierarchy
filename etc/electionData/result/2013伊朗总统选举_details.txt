'''2013年伊朗总统选举'''是[[伊朗伊斯兰共和国]]第11届总统选举，于2013年6月14日星期五举行。<ref name=" man" >{{cite web |url=http://nandu.oeeee.com/nis/201305/08/47717.html |title= 伊朗总统选举拉开大幕|accessdate=013-05-08 |publisher=南都网 |work= |language=zh-cn}}</ref>本次选举报名人数686人，<ref name="tr " >{{cite web |url=http://world.people.com.cn/BIG5/n/2013/0512/c57507-21453007.html |title=伊朗总统竞选报名人数达到686人 |accessdate=2013年5月12日 |publisher= 人民网|work= |language=zh-hk}}</ref>这次总统选举5人来自强硬保守派。<ref name="fhs " >{{cite web |url=http://news.ifeng.com/gundong/detail_2013_05/23/25610465_0.shtml |title=伊朗总统选举候选人名单公布 |accessdate=2013年5月29日 |publisher=凤凰网 |work= |language=zh-cn}}</ref>获资格审查通过共8人，最终六人参选。
本次大选的选民投票踊跃，投票所大排长龙，伊朗内政部还延长投票时间至夜间。6月15日晚内政部宣布，[[哈桑?鲁哈尼]]（改革派）获18,613,329票（得票率50.71%）因得票过半而胜出，第二名[[德黑兰]]市长[[穆罕默德?巴吉尔?卡利巴夫]]（强硬派）获得6,077,292张票。
地方议会选举也在同一天举行，由[[宪法监护委员会]]负责监督，该委员会由12人组成，包括6名[[法学家]]和6名[[神学家]]，他们负责审查候选人合格和确认选举结果。

1	背景	2	0	[2]
[[伊朗总统]]是由伊朗[[公民]]直接投票选举产生，每4年选举一次，伊朗总统是世俗事务的最高执行者，行使行政权，位置排名于伊朗革命和宗教领袖之后。<ref name="fish " >{{cite web |url=http://opinion.huanqiu.com/opinion_world/2013-05/3981128.html |title=德安：掀起伊朗总统选举的“面纱” |accessdate=2013年5月29日 |publisher=环球时报 |work= |language=zh-cn}}</ref>所有出生在伊朗的伊朗公民、信仰[[真主阿拉]]和官方宗教伊朗[[什叶派]][[伊斯兰教]]、忠于[[伊朗宪法|宪法]]、年满16周岁、75岁以下的可以作为候选人。<ref name="fish " />宪法监护委员会将根据伊朗宪法条款，审查申请参选者的[[政治]]和[[宗教]]背景，确定其参选资格。伊朗宪法要求，如果参选者超过200名候选人，宪法监护委员会将评估这些候选人并选择最优秀的担任伊朗总统。伊朗总统候选人有一个标准表单（[[教育]]、[[公共服务]]记录、[[军事]]服务记录等）都被列入表单。
